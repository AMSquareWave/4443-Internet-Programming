## Complete (Kinda) Site

### Goal: 
- Create a front end that uses a common css component library (aka bootstrap)
- Grab data from an api (backend) that will dynamically change our front end based on user input. 

### Tools:
- Python Flask
- Bootstrap css framework
- Html
- Javascript (jquery)
